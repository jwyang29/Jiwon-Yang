[ .Overview ]

Sound → Motion Graphics via Machine Learning

IML Workstation uses Interactive Machine Learning to map live audio features — pitch, loudness, timbre — to parameters of a 3D generative scene built in Three.js. Users train the mapping themselves, by example, in real time.

[ .Process ]

A Pipeline for Embodied Control

Audio is captured via Web Audio API and analyzed for spectral features using a custom FFT pipeline. Features are sent over OSC to Wekinator, which runs a k-NN regression model in real time.

Wekinator's output — a set of 0–1 parameters — is routed back into the browser and mapped to Three.js shader uniforms: particle count, orbit radius, color temperature, rotation speed.

✳
[ .Context ]

Design Research Through Making

Developed in the Interactive Media Lab (IML) at SNU as a research prototype exploring embodied interaction with generative systems. User testing revealed that non-expert participants found the training loop legible within 90 seconds — and spent significantly longer with the system than with a fixed-mapping equivalent.
