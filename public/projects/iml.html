Overview

Sound → Motion Graphics via Machine Learning

IML Workstation is a browser-based creative tool that uses Interactive Machine Learning (IML) to map live audio features — pitch, loudness, timbre — to parameters of a 3D generative scene built in Three.js.

Users train the system in real time: hum a low note → scene contracts. Whistle high → particles scatter. The mapping is personal, intuitive, and entirely up to the user. The machine learns what the body already knows.

System Design

A Pipeline for Embodied Control

Audio is captured via Web Audio API and analyzed for spectral features using a custom FFT pipeline. Features are sent over OSC to Wekinator, which runs a k-NN regression model in real time. Wekinator's output — a set of 0–1 parameters — is routed back into the browser and mapped to Three.js shader uniforms: particle count, orbit radius, color temperature, rotation speed.

The design challenge was not the pipeline but the interface: how do you let a non-technical user understand that they are training a model without exposing the model?

Outcome

Design Research Through Making

The project was developed in the Interactive Media Lab (IML) at SNU as a research prototype exploring embodied interaction with generative systems. User testing revealed that non-expert participants found the training loop legible within 90 seconds — and spent significantly longer with the system than with a fixed-mapping equivalent.
